## ) **Shoo Loong Kan – 火锅面 (Hot Pot Noodles)**

### **Nom du plat**

- **Shoo Loong Kan Hot Pot Noodles** (nouilles style fondue chinoise / hot pot)

### **Saveur**

- On lit **“麻辣牛油味”** = **saveur mala au gras de bœuf**
  - **麻辣 (málà)** = piment + poivre du Sichuan (piquant + picotement)
  - **牛油** = graisse / huile de bœuf (goût plus riche)

### **Recette / préparation (visible sur le pot)**

On voit les pictos sur le côté :

1. **Ajouter les sachets** dans le bol
2. Remplir avec de l’eau chaude (il y a une **ligne d’eau** / niveau)
3. Fermer
4. Attendre
5. Mélanger

### **Temps à l’eau frémissante**

- Sur ce type de bol : **5 minutes** (très probable, c’est ce qu’on voit souvent et ça ressemble à ce qui est imprimé)

### **Sachets : avant ou après ?**

Pour ce type **hot pot noodles**, généralement :

- **Avant l’eau** :
  - sachet poudre
  - légumes
  - base pimentée
- **Après (optionnel selon texture voulue)** :
  - huile piment / sauce huileuse, si tu veux garder plus d’arôme en surface
    *(mais beaucoup mettent tout avant, ça marche aussi)*

✅ Si tu veux plus de goût : mets **la moitié de l’huile avant**, **la moitié après**.
